# 命令

| 命令 | 简写 | 描述 |
|------|------|------|
| `/postdrop help` | `/pd help` | 显示帮助信息 |
| `/postdrop toggle` | `/pd toggle` | 切换物品丢弃保护状态 |
| `/postdrop reload` | `/pd reload` | 重载插件配置 |
| `/postdrop version` | `/pd version` | 显示插件版本信息 |
| `/postdrop update` | `/pd update` | 检查插件更新 |
