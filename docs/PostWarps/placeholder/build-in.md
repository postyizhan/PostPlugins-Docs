# 内置占位符

## 地标信息

- `{warp_name}` 正在编辑/查看的地标名称
- `{warp_owner}` 正在查看的地标的所有者
- `{warp_world}` 正在查看的地标的所在世界
- `{warp_count}` 地标数量
- `{warp_desc}` 正在编辑/查看的地标的描述
- `{x}, {y}, {z}` - 地标的坐标

## 菜单信息

- `{menu_name}` 菜单名称
- `{menu_title}` 菜单标题
- `{menu_icon}` 菜单图标
- `{menu_sub_icon}` 菜单子图标
- `{current_page}` - 当前页码
- `{total_pages}` - 总页数
- `{search_display}` - 搜索关键词
